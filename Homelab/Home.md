# Home

This wiki documents the homelab setup running on a Proxmox host with multiple VMs. It covers infrastructure configuration, setup steps, runbooks for common tasks, and decisions made along the way.

## Structure

- **Homelab** - all homelab documentation
  - **Architecture Overview** - high level overview of the setup
  - **Network Diagram** - visual representation of the network
  - **Setup** - chronological record of how everything was built
  - **Networking** - IP addresses, domains, ports, DNS
  - **Security** - authentication, certificates, network isolation
  - **Runbooks** - step by step procedures for common tasks
  - **Decisions** - records of why each tool was chosen
  - **Troubleshooting** - known issues and how they were resolved
  - **Reference** - quick lookup tables and useful commands
